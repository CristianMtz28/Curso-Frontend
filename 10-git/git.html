Git y Github

Git

Git es una herramienta que nos permite administrar nuestro código para que tengamos versiones de nuestro sitio web y así en el futuro podamos revisar el código que teníamos en el pasado y también nos ayuda a trabajar con más desarrolladores. Git es el más popular y es uno de los más fáciles pero hay más controlador de versiones

Github

Este es "la red social" del código ya que es una plataforma en la que subes tu código y los demás desarrolladores pueden ver y trabajar en tu código siempre y cuando quieras

Instalación

Debes entrar a https://git-scm.com/ y lo instalas como cualquier otro programa. Puedes verificar la instalación desde cmd y poniendo git; también puedes usar la git bash
