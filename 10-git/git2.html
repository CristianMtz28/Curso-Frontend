Historial

Revisando cambios en el historial de git

Para empezar veremos el comando:

git log

git log nos permitirá traer todos los commits que hemos hecho

Para ver un poco más a detalle qué archivos se modificaron y cuánas "inserciones" o líneas de código se modificaron usamos el comando:

git log --stat

Que nos muestra algo así como las estádisticas de los cambios

Cuando cambiamos nuestro código y añadimos y borramos una línea de código o más nos aparece algo así:

git2.html | 2 +- 1 file changed, 1 insertion(+), 1 deletion(-)

Para ver las diferencias en consola entre el commit pasado y la versión que tenemos ahora usamos el comando:

git diff

Para ver nuestros commits en una sola línea sin tanta información y sólo el mensaje usamos el comando:

git log --oneline

PARA REGRESAR A UNA VERSIÓN ANTERIOR USAMOS git checkout #### y el número de identificador; podemos regresar a él, pero si regresamos YA NO EXISTIRÁN LAS VERSIONES POSTERIORES ENTONCES DEBEMOS TENER MUCHÍSIMO CUIDADO
