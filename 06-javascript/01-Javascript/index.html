Javascript

Como has estudiado Javascrpt sí es un lenguaje de programación y esto nos puede ayudar tanto en el backend como en el frontend; tendremos este "repaso" completo y lo acabaremos para poder seguir con nuestra meta de manejar Angular y/o React, así que empecemos.

Javascript se puede usar dentro del archivo HTML con la etiqueta script y lo que haya dentro de la etiqueta junto con su cierre será ejecutado, sin embargo es mejor usar un archivo aparte con nuestras funcionalinades y llamarlo al archivo principal de index.html; así como lo hacíamos con CSS; para eso usaremos el script antes de nuestro body
